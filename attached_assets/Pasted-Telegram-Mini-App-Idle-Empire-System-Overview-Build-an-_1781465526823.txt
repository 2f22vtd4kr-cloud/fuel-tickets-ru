Telegram Mini App – Idle Empire System

Overview

Build an idle empire game inside our existing Telegram Mini App.

The primary purpose is retention and increasing daily active users.

Users already earn XP from purchases elsewhere in the platform. This game uses that XP as a spendable resource to develop an empire.

IMPORTANT:

A user’s account XP and account level must NEVER decrease.

The game has its own “available XP balance” that can be spent.

Example:

* User earns 10,000 XP through purchases.
* User account level remains permanently based on total lifetime XP earned.
* User can spend 2,000 XP inside the Empire game.
* Available XP balance becomes 8,000.
* Lifetime XP and user level remain unchanged.

Think of XP as:

* Lifetime XP = progression/history (never decreases)
* Available XP = spendable currency for Empire upgrades

⸻

Core Gameplay Loop

1. User earns XP through purchases.
2. XP appears in Empire wallet.
3. User spends XP to build structures.
4. Structures generate Coins passively.
5. Coins are collected manually or automatically.
6. Coins unlock upgrades and expansion.
7. User returns daily to collect rewards and continue progression.

Game should be playable in short sessions (30 seconds to 3 minutes).
